# Bulker

Третья пытка!

Необходимо установить **[Go](https://go.dev/dl/)**.

## Сборка

```sh
npm run build
```
Либо:
```sh
make build
```

## Запуск

```sh
npm run dev & go run . 
```
Либо:
```sh
make
```

## Тесты

Сервер запускается по адресу `localhost:8000`. Можно добавить в путь параметр `name`, т.е. `http://localhost:8000?name=Montferrat`
